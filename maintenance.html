Регламентные работы

Эта процедура предотвращает скопление грязи и неприятных запахов:

  1. Извлеките поддон из кофемашины.
  2. Вылейте остатки жидкости, если они есть.
  3. Промойте поддон тёплой водой с мылом, используя губку.
  4. Высушите поддон перед установкой на место.

Регулярная очистка фильтра предотвращает засоры и сохраняет вкус кофе:

  1. Снимите фильтр с помощью специального ключа, если он предусмотрен.
  2. Удалите остатки кофейной гущи с помощью щётки или мягкой тряпки.
  3. Промойте фильтр под струёй тёплой воды.
  4. Дайте фильтру высохнуть перед установкой обратно в кофемашину.

Для обеспечения оптимальной работы кофемашины выполняйте следующие шаги:

  1. Извлеките резервуар для воды из кофемашины.
  2. Слейте остатки воды, если они есть.
  3. Промойте резервуар теплой мыльной водой, используя мягкую губку.
  4. Тщательно ополосните резервуар под проточной водой, чтобы удалить остатки моющего средства.
  5. Высушите резервуар мягким полотенцем перед установкой обратно.

Процедура удаления накипи необходима для продления срока службы вашей кофемашины:

  1. Приготовьте раствор для удаления накипи, следуя инструкции производителя.
  2. Залейте приготовленный раствор в резервуар для воды.
  3. Запустите процесс очистки, выбрав соответствующий режим на панели управления кофемашины.
  4. После завершения программы слейте остатки раствора из резервуара.
  5. Пропустите через систему 2-3 резервуара чистой воды, чтобы удалить остатки средства.

Для предотвращения утечек проверяйте состояние уплотнителей:

  1. Откройте панель доступа, если это предусмотрено конструкцией.
  2. Осмотрите уплотнители на наличие трещин или износа.
  3. При необходимости замените повреждённые уплотнители, следуя инструкции производителя.
